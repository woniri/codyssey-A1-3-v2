# 📖 AI 기반 여행 스토리텔링 전자책 플랫폼 기획서
> **"비용 걱정 없이 만드는 나만의 AI 여행 오디오 가이드북 및 페이퍼 콜라주 e-Book 뷰어"**

이 기획서는 사용자가 가고 싶은 여행지를 입력하면 무료 AI 기술 및 오픈 API들을 유기적으로 조합하여 **무광 페이퍼 콜라주 감성의 전자책(e-Book)**과 **최적의 타임라인 추천 일정**을 자동으로 구성하고, Git과 Vercel을 통해 평생 무료 배포가 가능하게 설계된 웹 플랫폼의 기획 사양서입니다.

---

## 1. 핵심 구현 가능성 및 비용 무과금 설계 (Free API Architecture)

누구나 비용 걱정 없이 창업 프로토타입이나 개인 포트폴리오로 운영할 수 있도록 100% 무료 티어를 제공하는 강력한 API 조합을 사용합니다.

* **스토리 텍스트 생성 (LLM)**: **Google AI Studio (Gemini 2.5 Flash)**
  * 개인 개발자 무료 한도 내에서 빠르고 영리하게 작동합니다.
  * 입력된 여행지의 사실적 정보를 파악하고, 사용자 선택 옵션(체리, 전문가, 아동 톤)에 맞춰 스토리 대본과 시간대별 일정을 JSON 형식으로 출력합니다.
* **사실 관계 실시간 검증**: **Gemini Web Search Grounding**
  * Gemini API가 답변을 짓기 전에 실시간 Google 검색 결과를 교차 검증하도록 설계해 왜곡(Hallucination) 현상을 차단합니다.
* **고화질 삽화 연동**: **Unsplash API**
  * 매월 5,000회 무료 이미지 조회를 활용하여, 책의 챕터별 영어 테마 키워드로 실제 촬영된 감성적인 풍경 사진을 1:1 자동 매칭해 삽입합니다.
* **오디오 내레이션**: **브라우저 Web Speech API**
  * 값비싼 서버 비용이 드는 유료 TTS를 대신하여, 크롬이나 사파리 등 접속자의 디바이스에 내장된 한국어 음성 합성 엔진(`speechSynthesis`)을 활용해 비용 한 푼 없이 텍스트를 목소리로 변환합니다.

## 2. 디자인 가이드라인 및 미학적 규칙 (Aesthetics Design System)

사용자가 제시한 **"종이 공예 콜라주 및 박물관/미술관 여행 포스터"** 테마를 웹 인터페이스와 이미지 리소스 전반에 걸쳐 구현합니다.

### 🎨 1. 시각적 핵심 무드 (Theme & Mood)
* **layered paper-cut collage**: 손으로 깎아 만든 듯한 레이어드 종이 컷팅 콜라주 느낌.
* **handcrafted paper diorama**: 무광 과슈(Gouache) 종이의 무광 텍스처와 은은하게 겹치는 그림자를 통한 입체적인 공간감 연상.
* **museum-quality travel poster**: 세련되고 시적인 한국적 에디토리얼 감성.

### 🎨 2. 색상 팔레트 (Color Palette - 서울 감성 절제 톤)
완전한 원색을 배제하고, 우아하게 바람에 씻긴 듯 톤다운된 종이 색면을 차용합니다.
* **한지 백색 (Background)**: `#FAF6F0` (눈이 편안한, 미세한 입자감이 맴도는 아이보리 종이 톤)
* **기와 회색 (Text & Border)**: `#3E403F` (딱딱한 디지털 검정색을 배제한 차분한 암회색)
* **빛바랜 단청 녹청색 (Primary Point)**: `#8AA399` (저채도의 은은한 민트/세이지 톤)
* **말린 살구빛 (Secondary Point)**: `#D6A28C` (부드러운 저채도 어스 톤 오렌지)
* **공기감 있는 담청색 (Accent)**: `#AEBCC4` (하늘과 물의 깊이를 은은하게 머금은 소프트 블루)

### 🎨 3. CSS/웹 마크업 구현 방식 (Layered CSS Coding)
* **종이 질감 패턴**: CSS 배경에 아주 미세한 노이즈(SVG Noise Filter)를 결합하여 무광 종이 느낌을 재현합니다.
* **겹친 그림자 (Multi-layered box-shadow)**:
  여러 장의 종이가 포개진 입체감을 위해, 넓고 부드럽게 퍼지는 다중 섀도우를 요소들에 적용합니다.
* **칼로 오린 듯한 경계**: 딱딱하고 완벽한 일직선의 테두리 대신 부드럽고 따뜻한 마감(`border-radius: 12px~24px`)을 부여하여 손맛을 표현합니다.

## 3. 스마트 책 분할 및 입력 폼 설계 (Structure & Input)

* **스마트 분할 (Book Splitting)**:
  * Gemini API 프롬프트를 통해 입력 대상의 성격과 규모를 판별합니다.
  * 대규모 도시 단위일 때는 옴니버스 시리즈(`SERIES`) 구성을 택하고, 단일 문화재/명소일 때는 장별로 긴밀히 쪼개진 단일권(`SINGLE`) 형태로 책을 구성합니다.
* **입력 필드**:
  * **여행지 (필수)**: 대상 도시 또는 명소명.
  * **가이드 스타일 (선택)**: 다정한 `체리 🍒` 톤, 격식 있는 `학술 전문가 🎓` 톤, 이해하기 쉬운 `아이용 동화 👶` 톤.
  * **여행 일정 (선택)**: 당일치기 / 1박 2일 / 2박 3일 (타임라인 일정 생성 범위 제어).
  * **관심 테마 (선택)**: 역사/문화, 맛집, 포토존, 자연 등 강조점 가중치 제공.

## 4. Vercel 서버리스 파이썬 API 아키텍처

프론트엔드(정적 파일들)와 백엔드(Python Serverless Function)를 한데 묶어 Vercel에서 하나의 호스트 이름 아래 동작하도록 설계했습니다.
* **엔드포인트**: `POST /api/generate`
* **역할**: 프론트엔드로부터 조건값 수집 ➡️ Gemini API 호출 (Structured JSON 반환) ➡️ Unsplash API를 경유하여 이미지 링크 획득 ➡️ 완성된 도서 데이터 최종 반환.
* **보안성**: Gemini API Key 및 Unsplash Access Key는 깃허브에 올리지 않고, Vercel 대시보드의 환경 변수(Environment Variables)로 등록해 외부 유출을 원천 방지합니다.
