# 📖 AI 기반 여행 스토리텔링 플랫폼 '북어 트립 (Book어 Trip) 🍒' 기획서
> **"한 번의 검색으로 무한한 여행 서재를 엮는 나만의 감성 AI 입체 팝업북 가이드 서비스"**

이 기획서는 사용자가 입력한 여행지 지역을 분석하여 대/소규모 스마트 서재(`Book Shelf`)로 자동 분할하고, 선택한 여행지에 맞춰 페이퍼 콜라주 일러스트와 팩트 기반 스토리텔링 전자책(e-Book), 오프라인 소장용 파일 다운로드 및 브라우저 보관함을 제공하는 혁신적인 감성 가이드 웹 플랫폼의 기술 사양서입니다.

---

## 1. 플랫폼 아키텍처 및 비용 Zero 설계 (Cost-Free Tech Stack)

개발자의 비용 한계를 돌파하기 위해 100% 무료 티어로 동작하도록 연동 체계를 지혜롭게 조율했습니다.

* **서재 판정 & 책 본문 생성 (LLM)**: **Google AI Studio (Gemini 2.5 Flash)**
  * 최신 초고속 경량화 모델 `gemini-2.5-flash`를 탑재하여 비용 한도 없이 빠르게 스토리를 엮습니다.
  * 서버 장애 및 트래픽 폭주 대응을 위해 OpenRouter(무료 모델), Groq, Cloudflare, NVIDIA NIM 등 총 7대 백업 공급망을 로테이션(Rotation) Failover 체계로 연결했습니다.
* **실시간 역사/팩트 체크**: **Gemini Web Search Grounding**
  * Gemini API가 답변을 짓기 전에 실시간 Google 검색 결과를 교차 검증하도록 설계해 왜곡(Hallucination) 현상을 차단합니다.
* **고품질 종이 공예 일러스트 공급 (표지 및 책꽂이)**: **Pollinations AI Image API**
  * 책의 표지와 책꽂이에 꽂힐 도서 표지에 수채화 디오라마 아트 키워드(`water color paper-cut diorama`)를 조합한 고유 일러스트를 매핑하여 시각적 화려함을 보장합니다.
* **본문 실제 관광지 풍경 매칭 (본문 한정)**: **Unsplash API**
  * 도서 본문(2~4장) 영역에는 실제 고화질의 관광지 사진을 매칭하여 신뢰도를 끌어올립니다.
* **오디오 내레이션**: **브라우저 내장 Web Speech API**
  * 브라우저의 한국어 음성 낭독 엔진(`speechSynthesis`)을 활용하여, 서버 트래픽이나 과금 없이 상냥하고 차분한 로컬 가이드 Cherry의 내레이션을 재생합니다.
* **배포 및 데이터 보존**: **GitHub + Vercel + LocalStorage**
  * 깃허브 무중단 연동 배포 및 로컬스토리지 보존을 통해 서버 인프라 유지 비용을 0원으로 실현했습니다.

## 2. 입력 화면 및 스마트 서재 분할 (User Flow & Input)

### 📝 1. 입력 화면 옵션 구성
* **여행지 (필수)**: 대상 도시 또는 명소명 (예: "대한민국", "서울")
* **가이드 톤앤매너**: `체리 🍒 (세련된 친근한 대화체)` / `전문 해설가 🎓` / `아동용 동화 👶`
* **일정 기간**: 드롭다운을 통해 당일치기부터 **최대 4박 5일**까지 유연하게 선택 가능.
* **관심 테마 (8개)**: 역사와 전설 🏛️, 맛집/카페 🍜, 인생샷 포토존 📸, 자연/힐링 🌲, 미술/건축 🎨, 쇼핑/기념품 🛍️, 야경/밤거리 🌙, 현지 체험 🏺

### 📚 2. 스마트 서재(Book Shelf UI) & 로테이션
1. **대규모 구역 검색 (국가, 대륙 등)**:
   * 1차 API인 `/api/search`가 이 구역을 대규모(`SERIES`)로 판정하고, 하위 도시 목록들 6~7권을 생성해 리턴합니다.
   * 프론트엔드는 나무 책꽂이 위에 한 번에 3~5권의 책 카드를 렌더링하며, 카드 표지 배경에는 AI 썸네일 일러스트가 입혀져 꽂힙니다.
   * 책꽂이 밑에 위치한 **`🔄 다른 도시/명소 보기`** 버튼을 누르면, 나머지 도서 리스트로 책꽂이가 교대로 롤링(로테이션)됩니다.
2. **일반 도시 및 명소 검색 (서울, 경주 등)**:
   * 1차 API가 단일 구역(`SINGLE`)으로 판정하면, 책꽂이 서재 화면을 띄우지 않고 **자동으로 즉시 가이드북 상세 생성 프로세스로 연결하여 다이렉트로 책을 열어 줍니다.**
3. **로컬 캐시 즉시 복원**:
   * 책을 생성하기 전 로컬스토리지 보관함을 탐색하여, 이미 제작했던 책을 다시 생성하거나 클릭했을 경우 **API 호출을 건너뛰고 0.1초 만에 캐시 데이터로부터 즉시 복원**하여 책을 펼쳐 줍니다.

## 3. 웹 디자인 시스템 및 미학적 규칙 (Design Tokens)

* **양면 와이드 전면 표지 (Spanned Cover)**:
  * 책을 열자마자 나오는 첫 스프레드(1p & 2p) 전체에 걸쳐 수채화 풍경 일러스트가 양면에 와이드하게 펼쳐지도록 좌우 절반씩 매칭해 노출합니다.
  * 3페이지(Left)에는 실제 본문 이미지, 4페이지(Right)에는 본문 스토리가 배치되는 완벽한 종이책 감성을 재현합니다.
* **배경 블러 오버레이**: 현재 책의 표지 일러스트 이미지를 뒷배경에 45px로 뿌옇게 깔아 은은한 입체감을 연상케 합니다.
* **색상 팔레트**: 한지 백색(`#FAF6F0`), 기와 회색(`#3E403F`), 단청 녹청색(`#8AA399`), 살구 샌드색(`#D6A28C`), 책꽂이 나무색(`#D3B499`), 경고 빨간색(`#D9534F`).
* **종이 엽서 질감 리터칭 (CSS Filter)**: 이미지 위에 CSS `sepia(12%)`, `contrast(92%)` 필터 효과를 오버레이하여 차갑고 딱딱한 디지털 경계를 무광 과슈 종이 단면처럼 부드럽고 따뜻하게 보정합니다.
* **반응형 3D 가변 뷰어**: 데스크톱에서는 가로 `85vw` 비율(`aspect-ratio: 16/9`)로 책이 열리고, 모바일에서는 1면 슬라이드 스와이프 레이아웃으로 동적 변환됩니다.

## 4. 고도화 신규 기술 사양 (New Features)

* **🗄️ 내 서재 보관함 관리**:
  * 생성 완료된 책 데이터를 로컬스토리지에 저장하여 재방문 시 `localStorage`에서 0.1초 만에 불러옵니다.
  * 보관함 카드 우측 상단에 삭제 단추(`×`)를 신설하여, 필요 없는 도서를 서재에서 간편히 삭제할 수 있습니다.
* **💾 가이드북 소장용 내보내기 (Download)**:
  * 책의 텍스트, 이미지 경로, 일정을 조합하여 한 장의 예쁜 인쇄용 **단일 포스터 HTML 파일**을 메모리에서 합성해 로컬 디렉토리로 다운로드합니다.
* **일정 누락 방지 짝수 보정**:
  * 전체 생성된 페이지의 수가 홀수이면 자동으로 '빈 백커버' 책장을 덧붙여 전체 루프 인덱스를 짝수로 보정하는 안전 알고리즘을 장착했습니다.
* **🔗 해시(#) 기반 서버리스 도서 공유**:
  * 생성된 책 데이터 전체를 Gzip(deflate) 방식으로 압축 후 Base64 문자열로 변환하여 URL 해시(Anchor) 값에 탑재합니다.
  * 공유를 받은 사용자는 API를 재호출하지 않고, 브라우저 단에서 이 압축을 해제하여 0.6초 만에 원본 책과 100% 동일한 책을 즉시 복원해 감상합니다.
* **🎨 리소그래프 감성 2도 인쇄 엽서 지도**:
  * 엽서 탭에는 방콕 야시장 등 도시의 무드를 담은 빈티지 2도 인쇄 AI 엽서 지도가 통째로 렌더링되며, 백엔드 병렬 스레드를 활용한 사전 생성 트리거를 적용해 로딩 지연이 완전히 사라졌습니다.
* **📱 모바일 반응형 완전 대응**:
  * 모바일 기기에서도 화면이 깨지거나 버튼이 겹치지 않도록, 모바일 전용 헤더 및 수직 스택형 콤팩트 오디오 제어 카드를 지원합니다.
