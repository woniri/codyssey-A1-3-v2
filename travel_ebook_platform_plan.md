# 📖 AI 기반 여행 스토리텔링 플랫폼 '북어 트립 (Book어 Trip) 🍒' 기획서
> **"한 번의 검색으로 무한한 여행 서재를 엮는 나만의 감성 AI 입체 팝업북 가이드 서비스"**

이 기획서는 사용자가 입력한 여행지 지역을 분석하여 대/소규모 스마트 서재(`Book Shelf`)로 자동 분할하고, 선택한 여행지에 맞춰 페이퍼 콜라주 일러스트와 팩트 기반 스토리텔링 전자책(e-Book), 오프라인 소장용 파일 다운로드 및 브라우저 보관함을 제공하는 혁신적인 감성 가이드 웹 플랫폼의 기술 사양서입니다.

---

## 1. 플랫폼 아키텍처 및 비용 Zero 설계 (Cost-Free Tech Stack)

개발자의 비용 한계를 돌파하기 위해 100% 무료 티어로 동작하도록 연동 체계를 지혜롭게 조율했습니다.

* **서재 판정 & 책 본문 생성 (LLM)**: **Google AI Studio (Gemini 2.5 Flash)**
  * 최신 초고속 경량화 모델 `gemini-2.5-flash`를 탑재하여 비용 한도 없이 빠르게 스토리를 엮습니다.
* **실시간 역사/팩트 체크**: **Gemini Web Search Grounding**
  * Gemini API가 답변을 짓기 전에 실시간 Google 검색 결과를 교차 검증하도록 설계해 왜곡(Hallucination) 현상을 차단합니다.
* **고품질 종이 공예 일러스트 공급**: **Pollinations AI Image API**
  * Unsplash의 무작위성 사진 노출 한계를 완전히 극복하기 위해, 사용자가 원하는 종이 디오라마 아트 키워드(`layered paper-cut collage, handcrafted paper diorama`)를 실시간 무료 생성 이미지 API와 조합하여 100% 싱크로율의 그림을 획득합니다.
* **오디오 내레이션**: **브라우저 내장 Web Speech API**
  * 유료 TTS 서비스 대신 브라우저의 한국어 음성 낭독 엔진(`speechSynthesis`)을 활용하여, 서버 트래픽이나 과금 없이 상냥하고 차분한 로컬 가이드 Cherry의 내레이션을 재생합니다.
* **배포 및 데이터 보존**: **GitHub + Vercel + LocalStorage**
  * 깃허브 무중단 연동 배포 및 로컬스토리지 보존을 통해 서버 인프라 유지 비용을 0원으로 실현했습니다.

## 2. 입력 화면 및 스마트 서재 분할 (User Flow & Input)

### 📝 1. 입력 화면 옵션 구성
* **여행지 (필수)**: 대상 도시 또는 명소명 (예: "항저우", "경주")
* **가이드 톤앤매너**: `체리 🍒 (세련된 친근한 대화체)` / `전문 해설가 🎓` / `아동용 동화 👶`
* **일정 기간**: 드롭다운을 통해 당일치기부터 **최대 4박 5일**까지 유연하게 선택 가능.
* **관심 테마 (8개)**: 역사와 전설 🏛️, 맛집/카페 🍜, 인생샷 포토존 📸, 자연/힐링 🌲, 미술/건축 🎨, 쇼핑/기념품 🛍️, 야경/밤거리 🌙, 현지 체험 🏺

### 📚 2. 스마트 서재(Book Shelf UI) 흐름
1. 사용자가 도시 단위(`경주`)를 입력하면, 1차 API(`/api/search`)가 이 구역이 대규모(`SERIES`)임을 판정하고 하위의 명소 도서 목록 3권을 설계해 선반 그래픽 위에 3D 책 카드로 꽂아 렌더링합니다.
2. 사용자가 선반에서 원하는 책 한 권을 고르면 2차 API(`/api/generate`)가 트리거되어 해당 책에 포커싱된 4페이지 분량의 심층 이야기와 시간대별 일정을 실시간 조리해 펼쳐 줍니다.

## 3. 웹 디자인 시스템 및 미학적 규칙 (Design Tokens)

* **배경 블러 오버레이**: 현재 책의 표지 일러스트 이미지를 뒷배경에 45px로 뿌옇게 깔아 은은한 입체감을 연상케 합니다.
* **색상 팔레트**: 한지 백색(`#FAF6F0`), 기와 회색(`#3E403F`), 단청 녹청색(`#8AA399`), 살구 샌드색(`#D6A28C`), 책꽂이 나무색(`#D3B499`).
* **종이 엽서 질감 리터칭 (CSS Filter)**: AI로 생성된 일러스트 이미지 위에 12%의 세피아 필터 및 종이 노이즈 텍스처 효과를 오버레이하여 차갑고 딱딱한 디지털 경계를 무광 과슈 종이 단면처럼 부드럽고 따뜻하게 보정합니다.
* **반응형 3D 가변 뷰어**: 데스크톱에서는 시원한 가로 `85vw` 비율(`aspect-ratio: 16/9`)로 책이 열리고, 모바일에서는 1면 슬라이드 스와이프 레이아웃으로 동적 변환됩니다.

---

## 4. 고도화 신규 기술 사양 (New Features)

* **🗄️ 내 서재 보관함 (Local Storage)**:
  생성 완료된 책 데이터를 로컬스토리지에 저장하여 재방문 시 `localStorage`에서 0.1초 만에 불러오는 복원 패널을 지원합니다.
* **💾 가이드북 소장용 내보내기 (Download)**:
  책의 텍스트, 이미지 경로, 일정을 조합하여 한 장의 예쁜 인쇄용 **단일 포스터 HTML 파일**을 메모리에서 합성해 로컬 디렉토리로 다운로드합니다.
* **일정 누락 방지 짝수 보정**:
  일정 페이지가 n장 덧붙을 때 3D 펼침면 양짝이 어긋나 잘리는 현상을 막기 위해, 전체 페이지 개수가 홀수이면 자동으로 '빈 백커버' 책장을 덧붙여 전체 루프 인덱스를 짝수로 보정하는 안전 알고리즘을 장착했습니다.
