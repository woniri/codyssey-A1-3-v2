## 2. 고도화 완료 사양 핵심 요약

### ① 스마트 서재(Book Shelf UI) 및 자동 다이렉트 뷰
* 검색 직후 1차 API인 `/api/search`가 지역 규모를 판정합니다.
* **대규모 광역 구역 (대한민국, 일본 등)**: 나무 책꽂이 선반 위에 3D 책 카드로 꽂아 렌더링합니다. 각 도서 카드에는 테마에 상응하는 AI 이미지 썸네일이 은은한 배경으로 노출됩니다.
* **다른 곳 보기 (Rotation)**: 6~7개의 도서 후보군 중 한 번에 3~5개만 꽂아두고, 하단의 `🔄 다른 도시/명소 보기` 버튼 클릭 시 다음 후보군 도서 목록으로 로테이션됩니다.
* **일반 도시 및 명소 (서울, 도쿄, 경주 등)**: 책꽂이 서재 단계를 자동으로 생략하고, 곧바로 본문 상세 생성 프로세스로 이어져 즉시 e-Book이 펼쳐집니다.

### ② 내 서재 보관함 관리 및 개별 삭제
* 로컬스토리지에 저장 공간을 개설하여 엮은 도서를 영구 저장하고, 우측 슬라이더 Drawer를 통해 언제든 즉각 재호출하는 보관함을 완성했습니다.
* 보관함 리스트 각 카드의 우측에 삭제(`×`) 버튼을 장착했으며, 이벤트 버블링 전파 차단(`evt.stopPropagation()`) 처리를 완비해 오작동 없는 카드 삭제 기능을 구현했습니다.

### ③ 양면 와이드 표지 리디자인 & 본문 Unsplash 실사 복원
* 책을 열자마자 나오는 첫 스프레드(1p & 2p) 전체에 걸쳐 수채화 풍경 일러스트가 양면에 와이드하게 펼쳐지도록 좌우 절반씩 매칭해 노출합니다.
* 3페이지(Left)에는 실제 본문 이미지, 4페이지(Right)에는 본문 스토리가 배치되는 완벽한 종이책 감성을 재현합니다.
* 책 본문(2~4장) 영역에는 AI 이미지의 엑박/오류 및 형태 오차를 방지하기 위해 **Unsplash 실제 고화질 관광지 사진**을 복원 매칭하여 신뢰도를 확보하였습니다.

### ④ 보관함(LocalStorage) 및 단일 파일 내보내기(Export)
* 엮어진 책을 다운로드할 때 텍스트와 이미지 경로, 시간표 일정이 단청색 에디토리얼 레이아웃 포스터 서식으로 자동 합성된 **오프라인 소장용 단일 HTML 파일 다운로드** 기능을 제공합니다.

## 3. GitHub 및 Vercel 무료 배포 가이드 (CI/CD)

Vercel의 파이썬 배포 환경(FastAPI)을 위해 수정한 `vercel.json` 기반의 배포 방법입니다.

1. **GitHub 업로드**:
   ```bash
   git add .
   git commit -m "feat: apply cabinet deletion, double cover layout, and rotation for Book어 Trip"
   git push origin main
   ```
2. **Vercel 연동 및 배포**:
   Vercel 대시보드에서 `woniri/codyssey-A1-3-v2` 저장소를 연동하고 환경 변수 `GEMINI_API_KEY`를 설정한 뒤 **Redeploy** 해 줍니다. Vercel이 수동 빌드 규칙에 맞춰 무사히 웹 서비스를 활성화합니다.
